Проверить, установлен ли у тебя Git Bash, очень просто. Давай сделаем это пошагово.

? Способ 1: Проверь через меню «Пуск»
Нажми Пуск (или Win)
Начни печатать:


1
Git Bash
Если появится значок:
??? Git Bash
Или Git for Windows
> Значит, установлен.

? Способ 2: Найди в папке установки
Открой проводник и перейди:



1
C:\Program Files\Git\bin\bash.exe
или



1
C:\Program Files (x86)\Git\bin\bash.exe
Если файл есть — значит, Git Bash установлен.

? Способ 3: Проверь в командной строке
Открой CMD или PowerShell и выполни:

cmd


1
git --version
Если ответ такой:



1
git version 2.xx.x.windows.1
— значит, Git установлен, а вместе с ним и Git Bash.

?? При установке Git ты автоматически получаешь Git Bash. 

? Способ 4: Запусти Git Bash напрямую
Нажми Win + R, введи:



1
"C:\Program Files\Git\git-bash.exe"
и нажми Enter

Если откроется чёрное окно с надписью:

user@PC_NAME MINGW64 ~
$

 Это и есть Git Bash — он работает!
