# 🛡️ БезопасныйПродукт — Telegram-бот для проверки состава

Бот помогает проверять пищевые добавки (Е-коды) на безопасность по странам: Россия, США, ЕС и др.

## 📸 Что умеет?
- ✅ Проверка текста на Е-коды
- 🖼 Распознавание фото (OCR)
- 🌍 Автоопределение страны по языку
- 📊 Статистика использования
- 🎯 Кнопки в Telegram
- 🧪 Поиск по красителям, консервантам, подсластителям
